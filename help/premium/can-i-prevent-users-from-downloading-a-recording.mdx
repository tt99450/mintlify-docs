---
title: "Can I prevent users from downloading a recording?"
description: "BigBlueButton does not block recording downloads. Browser controls, network capture, and screen recording each give a determined user a way around any restriction."
---

Premium Tier viewers can right-click on a playing recording and select **Save video as…** to download it. BigBlueButton does not block this action.

The download menu is a function of the browser. Disabling it with JavaScript would not stop a determined user from downloading the recording: they could inspect network traffic to find the underlying URL, or screen-record the playback into a separate video file.

For these reasons, we have not added a restriction on **Save video as…** in BigBlueButton.
