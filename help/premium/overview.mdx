Blindside Networks offers two tiers of hosting: 

- **Essentials**: the essential capabilities to use BigBlueButton, and 
- **Enhanced**: an enhanced version of BigBlueButton with many value added capabilities that we (Blindside Networks) provide to ensure you have the highest learning outcomes.

The table below summarizes the differences.

| Feature | Essentials | Enhanced |
| --- | --- | --- |
| Max simultaneous virtual classes | 5 | Purchased |
| Students can see each other's webcams in a session | - | Yes |
| How long do recordings last? | 7 days | Permanent |
| Recordings are downloadable as video files | - | Yes |
| Recording links are sharable | - | Configurable |
| Ability for teacher to invite a guest into a session | - | Yes |
| Automatically share recordings with Canvas Studio | - | Yes |
| Analytics from session available later in recordings | - | Yes |
| Enterprise Learning Analytics Dashboard (ELAD) available, as upgrade, for insight across all your virtual classes. | - | Yes |
| Custom defaults (\*) | - | Yes |

(\*) See [Account customization](/help/premium/overview-of-premium-features#account-customization) for the full list of configurable defaults.

For a detailed walk-through of each Enhanced feature, see the [Overview of BigBlueButton Enhanced](/help/premium/overview-of-premium-features).
