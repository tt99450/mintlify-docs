## By default, the first moderator in the session is made the presenter.

Any moderator can make any user (including themselves) the presenter. The presenter role enables the assigned user to [upload a presentation](/help/presenter-guide/upload-a-presentation-enable-for-download), share a [link from YouTube](/help/presenter-guide/share-an-external-audiovideo-link), or [engage a poll](/help/presenter-guide/use-pollingquzzing).

![givepresenter.gif](/images/givepresenter.gif)

Click on the name of the user in the Users list and select **Make presenter**.

To take back presenter status, click on your name and select **Take presenter role**.

![takepresenter.gif](/images/takepresenter.gif)

Moderators can also take presenter status by selecting the action button and selecting **Take presenter role**.

![mceclip0.png](/images/mceclip0_15.png)

If you are not a moderator, you need to request that one of the moderators in the session make you presenter.
