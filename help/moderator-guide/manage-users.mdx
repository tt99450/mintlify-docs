## Moderators can manage settings for all users or individually.

All users (Moderators and Viewers) appear in the _User list_ panel on the left side. The User list can be displayed or hidden via the _User list toggle_ button.

<Frame>
  ![Screenshot 2026 03 16 At 7 35 43 PM](/images/Screenshot2026-03-16at7.35.43PM.png)
</Frame>

To display the configuration menu with settings for all users, press the **Manage users** cog wheel beside **USERS.**

<Frame>
  ![Screenshot 2026 03 16 At 7 36 01 PM](/images/Screenshot2026-03-16at7.36.01PM.png)
</Frame>

### Manage users menu

<Frame>
  ![Managemenu](/images/managemenu.png)
</Frame>

**[1] Enable (or Disable) Users join muted** - dictates whether users joining the session will have their microphones muted or unmuted.

**[2] Mute all users except presenter** - mutes every user (Moderators and Viewers) except the presenter. Users can still unmute themselves.

**[3] Lock viewers** - opens another menu with more granular options. See the following [article](/help/moderator-guide/lock-viewers-ability-to-use-their-micwebcamchatshared-notes) for more information.

**[4] Guest policy** - sets the session user access policy. See the following [article](/help/moderator-guide/how-to-configure-guest-policy) for more information.

**[5] Save user names** - downloads the list of users in a text file.

**[6] Clear all reactions** - removes all emojis and raised hands.

**[7] Create breakout rooms** - opens the Breakout Rooms management interface. See the following [article](/help/moderator-guide/create-and-manage-breakout-rooms) for more information.

**[8] Learning Analytics Dashboard** - see the following [article](/help/moderator-guide/learning-analytics-dashboard) for more information.

### To display the configuration menu with settings for individual users, click on their name in the _User list_

<Frame>
  ![Studentmenu 1](/images/studentmenu-1.png)
</Frame>

**[1] Start a private chat** - opens a new window visible only to the Moderator and selected user for private messaging.

**[2] Lock public chat** - prohibits the user from posting messages in the _Public Chat_ window.

**[3] Mute user** - mutes the microphone of the selected user.

**[4] Give whiteboard access** - permits the selected user to mark up the whiteboard.

**[5] Make presenter** - promotes user to the [presenter](/help/presenter-guide/use-the-presentation-controls) role.

**[6] Promote to moderator** or **Demote to viewer** - configures user role.

**[7] Remove user** - ejects a user from the session and presents the option to _Prevent this user from rejoining the session_.
