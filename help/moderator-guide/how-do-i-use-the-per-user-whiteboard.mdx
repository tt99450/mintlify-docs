---
title: "How do I use the per-user whiteboard?"
description: "Moderators can grant whiteboard access to one viewer at a time, and remove access for an individual or all viewers at once."
---

Moderators can grant a single viewer whiteboard access without enabling it for the whole class — useful when you want one student to mark up the slide. You can [grant access to a user](#grant-whiteboard-access-to-a-user) and [remove access](#remove-whiteboard-access) at any time.

For full whiteboard mode, see the [multi-user whiteboard](/help/releases/how-do-i-use-the-multi-user-whiteboard).

## Grant whiteboard access to a user

<Frame>
  ![Whiteboardaccess](/images/whiteboardaccess.png)
</Frame>

To grant access, select a user in the **Users list** **[1]**, then select **Give whiteboard access** **[2]**.

A pen icon appears next to the user's avatar to confirm access.

<Frame>
  ![Screenshot 2026 03 16 At 11 05 11 PM](/images/Screenshot2026-03-16at11.05.11PM.png)
</Frame>

That user can now use all the [multi-user whiteboard](/help/releases/how-do-i-use-the-multi-user-whiteboard) tools.

A counter appears on the multi-user whiteboard icon showing how many users currently have access.

![Whiteboard access counter](/images/12233691486861.png)

## Remove whiteboard access

There are two ways to remove access.

<Frame>
  ![Screenshot 2026 03 16 At 11 05 37 PM](/images/Screenshot2026-03-16at11.05.37PM.png)
</Frame>

To remove access for everyone except the presenter, select the **multi-user whiteboard** icon **[1]**.

<Frame>
  ![Screenshot 2026 03 16 At 11 06 00 PM](/images/Screenshot2026-03-16at11.06.00PM.png)
</Frame>

To remove access from a single user, select their name in the **Users list**, then select **Remove whiteboard access**.
