The [multi-user whiteboard](/help/releases/how-do-i-use-the-multi-user-whiteboard) is a popular feature in BigBlueButton. In BigBlueButton 2.3, the Per-User Whiteboard was added to help with user engagement and allow instructors to choose an individual user to mark up the presentation.

<Frame>
  ![Whiteboardaccess](/images/whiteboardaccess.png)
</Frame>

To enable the Per-User Whiteboard, select a user **[1]** and then select Give whiteboard access **[2]**.

Once you have given an individual user whiteboard access, a pen icon appears next to their avatar.

<Frame>
  ![Screenshot 2026 03 16 At 11 05 11 PM](/images/Screenshot2026-03-16at11.05.11PM.png)
</Frame>

The user now has all the capabilities outlined in the [multi-user whiteboard](/help/releases/how-do-i-use-the-multi-user-whiteboard) article.

When granting individual whiteboard access, a count will appear on the multi-user whiteboard icon showing you how many students you have granted access to the whiteboard.

![mceclip2.png](/images/12233691486861.png)

## How do I remove whiteboard access once it has been granted?

There are two ways to remove whiteboard access.

<Frame>
  ![Screenshot 2026 03 16 At 11 05 37 PM](/images/Screenshot2026-03-16at11.05.37PM.png)
</Frame>

Clicking the multi-user whiteboard icon **[1]** removes whiteboard access from everyone except the presenter.

<Frame>
  ![Screenshot 2026 03 16 At 11 06 00 PM](/images/Screenshot2026-03-16at11.06.00PM.png)
</Frame>

You can also remove individual users' access by clicking their user name from the list of users and then clicking **Remove whiteboard access.**
