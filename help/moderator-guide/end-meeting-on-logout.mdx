---
title: "End meeting on logout"
description: "Moderators can end the session for everyone on logout, which closes the meeting and sends recordings for processing immediately."
---

Any user can log out of BigBlueButton, but only a moderator can end the session for everyone on logout. Ending the session ensures the meeting does not stay open if other users have not logged out, and it sends any recorded segments for processing immediately.

## End the session for all users

Select the **Leave** button, then select **End session**.

A final message confirms the session has ended. Select **OK** to return to your institution's home page.
