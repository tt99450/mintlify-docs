## Overview

When a BigBlueButton conference or activity is enabled for recording, you can use the Start/Stop button in the live session to generate a recording.

It is important to note that enabling your conference or activity to be recorded when it was created is **not** an auto-capture function. You **must** still use the start/stop record button in the live session for a recording to process.

![Screen_Shot_2019-02-27_at_10.42.55_AM.png](/images/Screen_Shot_2019-02-27_at_10.42.55_AM.png)

The recording icon is located in the top centre of the BigBlueButton interface to the right of the session title.

## Start the recording

To start the recording, select the **Start Recording** button:

![start_recording.gif](/images/start_recording.gif)

When started, the recording icon will change to red and a timer will begin counting up the total number of recorded minutes captured.

## Stop or pause the recording

To stop or pause the recording, select the **red icon and timer**:

![stop_recording.gif](/images/stop_recording.gif)

<Note>The recording processing takes approximately the same amount of time as the **full** duration of the live session and does not begin until the session has ended or all attendees have logged out of BigBlueButton.</Note>

<Note>All sessions have a duration of 9 hours.</Note>
