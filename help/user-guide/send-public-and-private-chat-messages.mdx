Anyone can send chat messages in BigBlueButton; you can send both [public](#to-send-a-public-chat) and [private chat](#to-send-a-private-chat) messages. You can also choose to edit, react, delete and reply to messages in the chat.

## Send a public chat message

To send a public chat message, select **[1] Public Chat** in the left panel, **[2]** type message, then select **Send message**.

![Image](/images/image-11.png)

<Accordion title="If the left panel isn't visible">
  Select the **User list toggle** button to open/close it.

  <Frame>
    ![Image](/images/image-12.png)
  </Frame>
</Accordion>

## Send a private chat message

To start private message to another user, **[1]** select the user with the mouse, **[2]** select **Start a private chat**.  A new private chat panel appears to the right.

![Image](/images/image-14.png)

In the new panel **[3]** type your message and select **Send message**.  Only that user will see you private messages.

![Image](/images/image-16.png)

Later, when the user responds, if the panel is not in focus a count of unread messages will appear messages appear next to their name.

## Chat message options

Moving your cursor over a chat message shows the message options bar with four options.

- Reply to a message
- Add an emoji reaction
- Edit a message
- Delete a message

### Reply to a message

You can reply to a message, which makes a copy of the original with your reply.

To reply to a message, move the cursor over the message and **[1]** select **Reply to message**.  The selected message appears above the message field, then **[2]** type reply and select [3] **Send message**.

![Image](/images/image-19.png)

The original message will appear again in the chat with your reply underneath for everyone to see.

![Image](/images/image-20.png)

### Add an emoji reaction

To add an emoji to a message, **[1]** select the React button from the message option menu, and **[2]** select an emoji from the pop-up, and then **[3]** the emoji will appear under the message.

![Image](/images/image-22.png)

Users can hover over the reaction to see who added it.

![Image](/images/image-23.png)

### Editing a message

To edit a message, **[1]** select **Edit** from the options bar, **[2]** Type in your new message and select **Send message**.

![Image](/images/image-24.png)

Your edit message will appear with the label _Edited._

### Deleting a message

To delete your message, select the **Delete message** button in the options bar.

![deletemessage.png](/images/33182999842445.png)

A prompt will appear to confirm that you want to permanently remove your message from the chat.

<Frame>
  ![deleteconfirm.png](/images/33183009026061.png)
</Frame>

<Note>
  Viewers can only edit and delete their own chat messages.  Moderators can delete any chat message.
</Note>
