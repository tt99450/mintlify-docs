Anyone can send chat messages in BigBlueButton; you can send both [public](#to-send-a-public-chat) and [private chat](#to-send-a-private-chat) messages. You can also choose to edit, react, delete and reply to messages in the chat.

## Send a public chat message

<Frame>
  ![Image](/images/image-11.png)
</Frame>

To send a public chat message, select **[1] Public Chat** in the left panel, \*_[2]_ type message, then select **Send message**.

If the left panel isn't visible, select the **User list toggle** button to open/close it.

<Frame>
  ![Image](/images/image-12.png)
</Frame>

## Send a private chat message

<Frame>
  ![Image](/images/image-14.png)
</Frame>

To start private chat with another user, **[1]** select the user, then **[2]** select **Start a prviate chat**.  A new private chat panel appears titled "Private Chat with \<user\>".

<Frame>
  ![Image](/images/image-16.png)
</Frame>

In the new panel, **[3]** type your private message and select **Send message**.  Only you and the other user will see the private chat messages.

When a user responds to your private message, you'll see a count of the unread messages appear next to their name.

<Frame>
  ![Image](/images/image-17.png)
</Frame>

## Chat message options

When sending a chat message, you have 4 ways that you can edit and interact with your messages.

Moving the cursor over a message shows an options bar that lets you

1. Reply to a message
2. Add an emoji
3. Edit a message
4. Delete a message

### Reply to a message

<Frame>
  ![Image](/images/image-19.png)
</Frame>

To reply to a message, move the cursor over the messag and **[1]** select **Reply to message**, then **[2]** type in your reply and select [3] **Send message**.

Your reply will appear with the original message linked in the chat.

<Frame>
  ![Image](/images/image-20.png)
</Frame>

### Emoji reactions

To react to a message, select the react button from the message option menu. An emoji menu will appear, and you can choose from a wide range of emojis to use. <img src="/images/33182999839245.png" alt="emojimessage.png" />

**Editing a message**

To edit a message, select the edit button. You can edit your message in the same chat box, and send the message to finalize your changes.

![editmessage.png](/images/33183009019533.png)

**Deleting a message**

![deletemessage.png](/images/33182999842445.png)

To delete your message, select the delete button and a prompt will appear to confirm that you want to permanently remove your message from the chat.

![deleteconfirm.png](/images/33183009026061.png)

_This feature will be available with the upcoming BigBlueButton 3.0 update._
