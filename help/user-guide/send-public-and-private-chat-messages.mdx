Anyone can send chat messages in BigBlueButton; you can send both [public](#to-send-a-public-chat) and [private chat](#to-send-a-private-chat) messages. You can also choose to edit, react, delete and reply to messages in the chat.

## **To send a Public Chat:**

![Screen_Shot_2019-02-11_at_4.28.06_PM.png](/images/360020987112.png)

The **Public Chat** **[1]** panel is located in the upper left-hand corner of the BigBlueButton interface.

![Screen_Shot_2019-02-11_at_4.32.26_PM.png](/images/360020915571.png)

If you do not see the public chat option available you may need to select the **Users List / Chat Toggle [2]**to expand the users list panel and public chat.

![Screen_Shot_2019-02-11_at_3.41.53_PM.png](/images/360020913231.png)

Select **Public Chat [1]** to expand the public chat panel.  To hide the public chat panel, select the **“\<” [2]** symbol.

Use the **Chat Input [4]** field to type messages into the public chat. Your chat messages (and messages from other users) will appear in chronological order.

You can save or download public chat by selecting **Chat Options [3]** menu.

![Screen_Shot_2019-02-11_at_3.42.04_PM.png](/images/360020913331.png)

When the public chat pane is open you can either **Save [1]** (you will be prompted to download the public chat as a text file), or **Copy [2]**(the public chat will be copied to the clipboard) the chat.

![Screen_Shot_2019-02-11_at_4.35.44_PM.png](/images/360020987292.png)

When minimized, the Chat indicator will appear when there are unread chat messages available.

## **To send a Private Chat:**

![Screen_Shot_2019-02-11_at_4.38.41_PM.png](/images/360020915891.png)

To private chat with another user, select their name (a pop-up menu will appear), choose**Chat [1]**.  A **Private Chat panel** will appear. To hide the panel, select either **“\<” [2]** or **“x” [3]**

Use the **Chat Input [4]** field to type messages into the private chat.

![Screen_Shot_2019-02-11_at_4.40.59_PM.png](/images/360020987572.png)

All public and private chat messages will appear under the Public Chat menu under **messages**.

![Screen_Shot_2019-02-11_at_4.43.38_PM.png](/images/360020916111.png)

As with the Public Chat, you will see a count of unread messages (highlighted in red) next to the user name.

## **Chat message options:**

When sending a message to the chat, you have 4 ways that you can edit and interact with your messages. An option menu will appear when you hover your cursor over a message. You can choose to reply to a message [1], emoji react to a message [2], edit a message [3], or delete a message [4].

## messageoptions.png

**Replying to a message**

![replyingmessage.png](/images/33183009014285.png)

To reply to a message, select the reply button and type your reply into the chat box below. Your reply will appear with the original message linked in the chat.

![replyresult.png](/images/33183009016717.png)

**Emoji reactions**

To react to a message, select the react button from the message option menu. An emoji menu will appear, and you can choose from a wide range of emojis to use. ![emojimessage.png](/images/33182999839245.png)

**Editing a message**

To edit a message, select the edit button. You can edit your message in the same chat box, and send the message to finalize your changes.

![editmessage.png](/images/33183009019533.png)

**Deleting a message**

![deletemessage.png](/images/33182999842445.png)

To delete your message, select the delete button and a prompt will appear to confirm that you want to permanently remove your message from the chat.

![deleteconfirm.png](/images/33183009026061.png)

*This feature will be available with the upcoming BigBlueButton 3.0 update.*
