---
title: "Join breakout rooms"
description: "As a viewer, join, rejoin, and use features within a breakout room."
---

In BigBlueButton, a moderator may create breakout rooms for small group discussions, collaboration, or team work; only moderators can engage breakout rooms.

If you enter a session after the breakout rooms have been launched, you can inform the moderator and they can resend an invitation for you to join.

## Join a breakout room

When breakout rooms are engaged, you receive a join prompt from BigBlueButton.

<Frame>
  ![Breakoutroominvitation](/images/breakoutroominvitation.png)
</Frame>

Select **Join** to enter the breakout room; selecting cancel returns you to the main session.

<Frame>
  ![Micselect](/images/micselect.png)
</Frame>

Your audio in the main session disconnects and you are prompted to connect your chosen audio format in the breakout room.

<Tip>
  We recommend joining with a **Microphone** so you can participate in the discussion.
</Tip>

<Frame>
  ![Mainandbreakout](/images/mainandbreakout.png)
</Frame>

The breakout room opens in a secondary tab; you can always return to the main session to communicate with the moderator if necessary.

## Use breakout room features

When you enter the breakout room, the interface looks the same as the main BigBlueButton session; however, there are a couple of important notes:

1. All users are connected to the breakout rooms with limited moderator status.
2. The first user to make a successful connection to the breakout room is allocated the presenter role.

The overall interface and important components remain the same.

<Frame>
  ![Breakoutroomlayout](/images/breakoutroomlayout.png)
</Frame>

Features in the breakout rooms consist of:

- **Breakout room title and time [1]:** indicates the current tab is running a breakout room session and the time remaining; the breakout rooms automatically close when this duration is met.
- **Whiteboard annotations [2]:** accessible by the user with the presenter status; enables annotations to the whiteboard or uploaded document.
- **Presenter actions menu [3]:** start a poll, upload a presentation, a link from YouTube, or take presenter status from the current presenter.
- **Screen sharing [4]:** accessible by the individual with presenter status; enables the individual to share their screen.

## Rejoin a breakout room

If you lose the connection or accidentally close the breakout room window, you can rejoin the room you originally connected to.

Return to the main session and navigate to the messages and users list panel.

Expand the **Breakout Rooms [1]** menu and select **Join Room [2]** to rejoin the breakout room.

<Frame>
  ![Joinroom](/images/joinroom-1.png)
</Frame>

If you do not see the breakout room menu, you can request the moderator resend a breakout room invitation to you.
