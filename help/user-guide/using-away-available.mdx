---
title: "Using away/available mode"
---

When entering a session, by default, your mode will be set to **available** in the right-hand options menu.

<Frame>
  ![Defaultavailable](/images/defaultavailable.png)
</Frame>

You can toggle the switch to set yourself to be **away**.

<Frame>
  ![Awaystatus](/images/awaystatus.png)
</Frame>

When you set yourself as away, the following happens:

- Your microphone is muted
- Your webcam is masked
- An "away" clock icon is added to your user avatar
- A note saying that you are away is added to the chat.

You can use the toggle switch to revert back to the **available** mode, which will reverse everything from being away.

<Frame>
  ![Availablestatus](/images/availablestatus.png)
</Frame>
