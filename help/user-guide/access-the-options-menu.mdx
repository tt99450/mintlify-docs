## Any user can access additional settings, information, and assistance through the options menu.

Located in the upper right-hand corner of the BigBlueButton interface, the options menu houses specific application settings and information for troubleshooting purposes.

<Frame>
  ![Options3dots](/images/options3dots.png)
</Frame>

To open the menu, select the three dots (hamburger menu) in the upper right-hand corner.

<Frame>
  ![Optionsmenu](/images/optionsmenu.png)
</Frame>

There are several options available within the Menu:

- **Away Status [1] updates user presence**
- **Make full screen [2]** enables full screen mode
- [**Settings [3] access additional application settings**](#settings)
- [**About [4] BigBlueButton copyright and build information**](#about)
- [**Help [5] access tutorial videos**](#help)
- [**Hotkeys [6] provides a list of hotkeys for quick keyboard navigation**](#hotkeys)
- [**Manage layout [7] displays presentation Layout options**](#manage-layout)

## Away Status

Setting to **away** will (a) pause your webcam, (b) mute your microphone, and (c) let everyone know in the chat you are away - all with a single click.

## Settings

The **Settings** menu houses several features that a user can enable or disable to customize their experience in the BigBlueButton meeting.

Subsections of the settings menu include the [**applications**](#application-menu) menu and [**data savings**](#data-savings) menu.

### Application menu

<Frame>
  ![Applicationmenu](/images/applicationmenu.png)
</Frame>

By default, the Application menu is expanded; a user can make changes to the BigBlueButton **Animations [1]** (enabled by default).

To change the default language, select from the list of available languages found in the **Application Language** **[2]** drop down.

You can also change the application **Font Size** **[3].** To make the font larger, select the “+” and to decrease the font size, select the **“-”.**

<Frame>
  ![Notifications](/images/notifications.png)
</Frame>

Enable **Audio alerts [4]**, and **Popup alerts [5]** to notify you of new incoming chat messages while in the BigBlueButton meeting and when your chat pane or users list is minimized.

**It is important to note that all alerts are in-browser only and do not behave like desktop push notifications.**

Select **Save** to apply any changes made or **Close** to return to the meeting.

### Data Savings

Settings found within the Data Savings menu are intended for users who are connecting from a mobile device or those connecting to networks performing under the recommended bandwidth requirements for optimal use of BigBlueButton.

![Screen_Shot_2022-08-29_at_3.52.05_PM.png](/images/8775950917901.png)

By default both the **Webcam [1]** and **Desktop Sharing [2]** streams are enabled.

![Screen_Shot_2022-08-29_at_3.47.49_PM.png](/images/8775848271885.png)

To conserve bandwidth, turn to **off**. When turned off any incoming/outgoing webcam or desktop sharing streams will not appear.

## About

The about section houses the BigBlueButton copyright information and build number.

![](/images/33953847081869.png)

You may be asked to provide the **Client** **build number** of the meeting you are connected to for troubleshooting purposes. It should appear under the copyright information.

## Help

The help button will redirect you to specific [help articles](https://bigbluebutton.org/tutorials/) that can assist with FAQs and other troubleshooting tips.
