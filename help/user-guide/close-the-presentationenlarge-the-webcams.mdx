---
title: "Close the presentation/enlarge the webcams"
description: "Any user can close the presentation area to enlarge shared webcams, or disable incoming webcams to save bandwidth."
---

Any user can [close the presentation](#close-the-presentation) to give shared webcams the full screen — useful for a video chat-style session. You can also [disable incoming webcams](#disable-incoming-webcams) to save bandwidth on a slow connection.

<Note>
  You can only close the presentation when at least one webcam is being shared.
</Note>

## Close the presentation

![Presentation area with shared webcams](/images/Screen_Shot_2019-03-04_at_3.31.21_PM.png)

To close the presentation, select the **X** in the upper-right corner of the presentation area. The webcam streams expand to fill the full BigBlueButton interface.

![Webcams enlarged after closing the presentation](/images/Screen_Shot_2019-03-04_at_3.43.21_PM.png)

To restore the presentation, select the **Presentation** icon in the lower-right corner.

<Note>
  There is no hard limit on the number of webcams that can be shared, but bandwidth is the practical limit. With many webcams, some may stop updating.
</Note>

## Disable incoming webcams

To turn off all incoming webcam streams — useful on mobile or low-bandwidth connections — open the **Settings** menu.

![Options menu showing Settings](/images/Screen_Shot_2022-08-31_at_10.04.28_AM.png)

Select **Data savings**, turn off **Enable other participants webcams**, then select **Save**.

![Data savings settings with webcams disabled](/images/Screen_Shot_2022-08-31_at_10.03.50_AM.png)
