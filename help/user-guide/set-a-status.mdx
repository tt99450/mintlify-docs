## Any user can set a status/emoji in BigBlueButton.

Use the status setting to indicate that you are away from the computer, happy, or confused, or use it to raise a hand for assistance.

## To set a status

### Option 1

![Screen_Shot_2019-02-12_at_1.38.09_PM.png](/images/360021126252.png)

Select the **Users List Toggle [1]** to expand the users list panel and public chat if it is not already visible.

![Screen_Shot_2019-02-12_at_1.38.20_PM.png](/images/360021126312.png)

Select your \*\*User Icon [2] \*\*in the users list (it will always be at the top).

![Screen_Shot_2019-02-12_at_2.12.33_PM.png](/images/360021126492.png)

Choose**Set a Status [3]** - A pop-up menu will appear listing the available emojis or status icons available to choose from the **Status List [5]**.

If you do not want to set a status, select \*\*Back [4] \*\*to return to the users list.

![Screen_Shot_2019-02-12_at_2.15.42_PM.png](/images/360021126632.png)

When you choose a status emoji it will appear in the centre of your \*\*User Icon \*\*in the users list. Other attendees will see your user icon move to the top of the users list.

### Option 2

Select the carrot (^) from the bottom righthand corner of the raise-hand emoji.

![](/images/17313432234893.png)

A pop-up menu will appear listing the available emojis or status icons available to choose from. Selecting one will set the corresponding status.

![](/images/17313616725517.png)

To clear your status, select your name in the\*\*Users list \*\* then select **Clear Status**.

Note: a moderator can also clear your status.

![Screen_Shot_2019-02-12_at_2.18.45_PM.png](/images/360021126772.png)
