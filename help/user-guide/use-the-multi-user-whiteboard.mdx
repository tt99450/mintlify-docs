## When the presenter enables the multi-user whiteboard, all users in the BigBlueButton session can annotate the whiteboard simultaneously.

![Screen_Shot_2019-02-21_at_11.05.55_AM.png](/images/Screen_Shot_2019-02-21_at_11.05.55_AM.png)

Once the presenter has engaged the multi-user capabilities, the whiteboard controls will appear to the right hand side in the presentation area.

![Screen_Shot_2019-02-21_at_11.16.01_AM.png](/images/Screen_Shot_2019-02-21_at_11.16.01_AM.png)

By default the **Pencil [1]**tool is enabled; you can use the other whiteboard controls to change the **T****hickness** **[2]**and **C****olour [3]** of the line.

![pencilexample.gif](/images/pencilexample.gif)

To draw on the presentation select and drag the mouse button to draw a line.

![Screen_Shot_2019-02-21_at_11.16.12_AM.png](/images/Screen_Shot_2019-02-21_at_11.16.12_AM.png)

You can **U****ndo last annotation** **[4]** made or **Clear all annotations [5]**while the multi-user whiteboard is enabled.

It is important to note that once the presenter turns the multi-user capabilities off they can clear the whiteboard marks made by others when it was enabled.

![Screen_Shot_2019-02-21_at_11.21.23_AM.png](/images/Screen_Shot_2019-02-21_at_11.21.23_AM.png)

Select the **Pencil** to display the other available whiteboard tools, which include:

- Text input **[2]**
- Line **[3]**
- Ellipse**[4]**
- Triangle **[5]**
- Rectangle **[6]**
- Pencil **[default]**
- Pan/pointer **[7]**

![textexample.gif](/images/textexample.gif)

To write on the presentation, select the **Text Input [2]** tool and draw a rectangle.  The rectangle will be the text area.
