---
title: "Use shared notes"
description: "Anyone in a session can contribute to the Shared Notes panel and export the notes to a file."
---

In addition to public and private chats, anyone in a BigBlueButton session can [contribute to Shared Notes](#contribute-to-shared-notes) and [export the notes](#export-the-notes-to-a-file) to a file. Shared Notes is a common text file that everyone in the session can edit at the same time.

## New Shared Notes Editor

In [March 2026](/product-updates#march-2026) we released a modern shared notes editor.  This is a markdown aware editor that provides support for markdown styles and tables.


## Contribute to Shared Notes

<Frame>
  ![Sharednotesuser](/images/sharednotesuser.png)
</Frame>

You can type '/' and see a pop-up menu of additional components to add.

## Legacy Editor

<Frame>
  ![Sharednotesuser](/images/sharednotesuser.png)
</Frame>

To open the notes panel, select **Shared Notes** **[1]** in the left sidebar. **Formatting Options** **[2]** such as **Bold** and **Italic** appear at the top of the panel. To reveal undo, redo, and download buttons, select the **'+'** **[3]** button.

When another user edits the notes, an alert appears on the **Shared Notes** icon.

## Export the notes to a file

To download the current notes, select the **Export** **[4]** button.

<Frame>
  ![Notesexportstudent](/images/notesexportstudent.png)
</Frame>

A list of available formats appears. Select your preferred format and save the file to your device.

<Frame>
  ![Notesnotification](/images/notesnotification.png)
</Frame>

<Note>
  - Shared Notes are not captured in the recording.
  - Each user must open their own Shared Notes panel.
  - Contributions to Shared Notes are anonymous.
</Note>
