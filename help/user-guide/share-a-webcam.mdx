Any user with a built-in or external webcam can [share their webcam](#share-your-webcam) so others in the session can see them. You can also [adjust webcam brightness](#adjust-webcam-brightness), [set a virtual background](#change-the-webcam-background), [pin a webcam](#pin-a-webcam), and [share multiple webcams](#share-multiple-webcams).

Configure your webcam device and set it as the default in your operating system before joining the session.

## Share your webcam

<Frame>
  ![Webcam icon in the BigBlueButton media bar](/images/Screenshot2026-03-16at2.27.17PM.png)
</Frame>

To start sharing, select the webcam icon in the media bar at the bottom of the BigBlueButton interface.

<Frame>
  ![Sharewebcam](/images/sharewebcam.png)
</Frame>

Your browser asks for permission to use the webcam. The prompt usually appears in the upper-right corner or the center of the browser. Select **Allow**.

<Frame>
  ![Browser webcam permission prompt](/images/Screenshot2026-03-17at1.19.49AM.png)
</Frame>

After allowing access, choose your device and quality settings.

<Frame>
  ![Websettings](/images/websettings.png)
</Frame>

If you have multiple webcams connected, select the one you want from the **dropdown** **[1]**. You can also set **Quality** **[2]**, **Brightness** **[3]**, and **Whole image** **[4]**, then select **Share** **[5]**. To set a virtual background, select **Virtual background** **[6]** in the next tab before sharing.

<Frame>
  ![Websettings2](/images/websettings2.png)
</Frame>

## Adjust webcam brightness

Use the **Brightness** slider at the bottom of the webcam settings window to change how bright your image appears.

If **Whole image** is selected, the slider adjusts the brightness of the entire image. If it is not selected, only your face is adjusted.

## Change the webcam background

BigBlueButton includes a set of default virtual backgrounds, and you can also upload your own.

To upload a custom background, select the camera icon, then **Upload**.

<Frame>
  ![Upload custom background option](/images/Screenshot2026-03-17at1.16.47AM.png)
</Frame>

Select an image file (`.jpg`, `.png`, or `.webp`) from your computer. The uploaded image appears in the first position on the left in the list of available backgrounds.

  src="https://mintcdn.com/testcompany-0e915c76/XvRmVVYnceDmgSFi/images/19159155670797.gif?s=f606c558ada8a48385df2cc441dd0e6a"

## Pin a webcam

You can pin one or more webcams so they always stay visible — useful when one webcam is showing sign language interpretation, for example.

![2.6\_-\_Webcam\_Pin.gif](/images/19159145783693.gif)

## Share multiple webcams

After sharing one webcam, share a second by opening the video options menu.

<Frame>
  ![Video options menu](/images/image-40.png)
</Frame>

Select **Open advanced settings**. In the webcam dialog, select a different webcam **[1]**, then select **Start sharing** **[2]**.

<Frame>
  ![Advanced webcam settings dialog](/images/image-42.png)
</Frame>

The second webcam appears alongside the first.

<Frame>
  ![Two webcams sharing at once](/images/image-43.png)
</Frame>

## Troubleshoot Chrome webcam permissions

If you see the message **"The webcam is in use by another application"** in Chrome, check that you have allowed webcam access.

<Frame>
  ![Chrome webcam in use error](/images/Screenshot2026-03-16at2.00.36PM.png)
</Frame>

The browser permission prompt usually appears in the upper-right corner. Select **Allow** and choose the device you want to share.
