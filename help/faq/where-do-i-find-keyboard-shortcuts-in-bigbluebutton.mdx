### Keyboard Shortcuts are located in the Options Menu.

<Frame>
  ![Opmenu](/images/opmenu.png)
</Frame>

To access the keyboard shortcuts open the **Options Menu** located in the upper right hand corner of the BigBlueButton User Interface.

<Frame>
  ![Keyshortmenu](/images/keyshortmenu.png)
</Frame>

Select **Keyboard Shortcuts.** 

## Keyboard shortcuts

The Keyboard shortcuts menu provides a listing of all the shortcut keys that can be used to navigate the BigBlueButton interface.

**Keyboard Shortcuts for Windows**
