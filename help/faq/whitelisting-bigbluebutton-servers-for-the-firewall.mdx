## BigBlueButton Servers

BigBlueButton servers use the following ports

| **Protocol** | **Port(s)** | **Description** |
| --- | --- | --- |
| TCP/IP | 80 | HTTP |
| TCP/IP | 443 | HTTPS |
| UDP | 16384-32768 | Encrypted WebRTC audio, video, and screen share media stream (DTLS-SRTP) |

## STUN/TURN Servers

- Outgoing and return UDP packets to port 3478 (STUN and TURN)
- Outgoing TCP connections to port 443 (TURN over TLS)
