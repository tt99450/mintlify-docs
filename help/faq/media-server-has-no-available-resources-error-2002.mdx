If when sharing a webcam you get the message **Media server has no available resources (error 2002)**, it means that the BigBlueButton server on which your meeting is running has currently reached its maximum capacity for webcam streams.

As a moderator, the solution is to try one of the following

1. Wait a few minutes and try again -- as other meetings with webcams on the server end, it will free up resources for webcams
2. Start a new meeting (create a new meeting link in your learning management system or Personal Rooms and share with users).Â  Using a new meeting link will ensure the meeting starts on a different server.
