---
title: "Antivirus software causes issues while connecting to a BigBlueButton Session"
description: "Resolve antivirus connection errors by temporarily disabling the Web Shield in Avast or similar antivirus software."
---

Users running Avast or similar antivirus software may see a connection error when joining a BigBlueButton session. Temporarily disabling the antivirus **Web Shield** restores access.

<Frame>
  ![Antivirus connection error shown when joining a BigBlueButton session](/images/Capture.PNG)
</Frame>

## Disable the Web Shield

To open the antivirus dashboard, launch your antivirus software.

<Frame>
  ![Avast antivirus dashboard](/images/2.PNG)
</Frame>

To open the protection settings, select **Protection** **[1]**.

<Frame>
  ![Protection icon highlighted in the Avast sidebar](/images/3.png)
</Frame>

To view the active shields, select **Core Shields** **[1]**.

<Frame>
  ![Core Shields option inside the Protection menu](/images/4.PNG)
</Frame>

To disable web filtering, select **Web Shield** **[1]** and switch it off.

<Frame>
  ![Web Shield toggle inside Core Shields](/images/5.PNG)
</Frame>

To choose the duration, select the option that matches how long the session will run.

<Frame>
  ![Duration options for disabling Web Shield](/images/6.PNG)
</Frame>

The **Web Shield** icon turns from green to red, confirming the shield is off.

<Frame>
  ![Web Shield icon shown in red after being disabled](/images/7.PNG)
</Frame>

<Tip>
  After the **Web Shield** is disabled, rejoin the BigBlueButton session. Re-enable the shield once the session ends to restore full protection.
</Tip>
