If when connecting to a BigBlueButton session, you encounter an error related to antivirus software, as attached in the screenshot below for Avast antivirus software, you will need to temporarily turn it off.

![Capture.PNG](/images/Capture.PNG)

Please follow the steps below to overcome the error: 

1. Open your antivirus software

![2.PNG](/images/2.PNG)

2. Click on **Protection** icon as highlighted below

![3.png](/images/3.png)

3. Inside the Protection, click on the **Core Shields**

![4.PNG](/images/4.PNG)

4. Select **Web Shield** and turn it off.

![5.PNG](/images/5.PNG)

5. Depending on the session requirement, select the option for how long you need to turn off the Web Shield.

![6.PNG](/images/6.PNG)

6. Once turned off, you will see the Web Shield icon turn to red from green.

![7.PNG](/images/7.PNG)

**Once the settings have been changed, you should be able to connect to the session.**
