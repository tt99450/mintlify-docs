To preserve shared notes from Breakout Rooms, select ***Save shared notes*** in the Breakout Rooms configuration panel.  When Breakout Rooms end, shared notes are converted to a PDF file, then become available as a Presentation from the action menu.

### 1. Open the Breakout Room configuration panel.  Select ***Create breakout rooms*** from the *Users* menu:

![](/images/34471808659597)

### 

### 2. Select ***Save shared notes***

![](/images/34471839974925)

### 3. After Breakout Rooms have ended, select Upload/Manage presentations

![](/images/34471839976589)

### 4. Enable download of the presentation

![](/images/34471839977101)

### 

### 5. Select the presentation file to download

![](/images/34471839977997)

### 6. Download the presentation

![](/images/34471839979021)
