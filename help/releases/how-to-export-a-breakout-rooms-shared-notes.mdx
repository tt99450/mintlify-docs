---
title: "How to export a Breakout Room's shared notes?"
description: "To preserve shared notes from Breakout Rooms, select Save shared notes in the Breakout Rooms configuration panel. "
---

To preserve shared notes from Breakout Rooms, select **_Save shared notes_** in the Breakout Rooms configuration panel. When Breakout Rooms end, shared notes are converted to a PDF file, then become available as a Presentation from the action menu.

### 1. Open the Breakout Room configuration panel. Select **_Create breakout rooms_** from the _Users_ menu:

![](/images/34471808659597.png)

### 2. Select **_Save shared notes_**

![](/images/34471839974925.png)

### 3. After Breakout Rooms have ended, select Upload/Manage presentations

![](/images/34471839976589.png)

### 4. Enable download of the presentation

![](/images/34471839977101.png)

### 5. Select the presentation file to download

![](/images/34471839977997.png)

### 6. Download the presentation

![](/images/34471839979021.png)
