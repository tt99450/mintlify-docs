title: "How to export a Breakout Room's whiteboard annotations?"
description: "To preserve the whiteboard annotations from Breakout Rooms, select Save whiteboard in the Breakout Rooms configuration panel. "
---

To preserve the whiteboard annotations from Breakout Rooms, select **_Save whiteboard_** in the Breakout Rooms configuration panel. When Breakout Rooms end, whiteboard annotations are converted to a PDF file, then become available as a Presentation from the action menu.

### 1. Open the Breakout Room configuration panel. Select **_Create breakout rooms_** from the _Users_ menu:

<Frame>

</Frame>

<Frame>

</Frame>

### 2. Select **_Save whiteboard_**

<Frame>
  ![](/images/34380386753037.png)
</Frame>

### 3. After Breakout Rooms have ended, select Upload/Manage presentations

<Frame>
  ![](/images/34413687728781.png)
</Frame>

  ![](/images/34413702325005.png)
</Frame>
