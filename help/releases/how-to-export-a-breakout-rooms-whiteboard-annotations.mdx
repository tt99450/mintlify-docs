To preserve the whiteboard annotations from Breakout Rooms, select ***Save whiteboard*** in the Breakout Rooms configuration panel.  When Breakout Rooms end, whiteboard annotations are converted to a PDF file, then become available as a Presentation from the action menu.

### 1. Open the Breakout Room configuration panel.  Select ***Create breakout rooms*** from the *Users* menu:

![](/images/34413687726477)

### 2. Select ***Save whiteboard***

![](/images/34380386753037)

### 3. After Breakout Rooms have ended, select ***Upload/Manage presentations*** from the *Actions* menu:

![](/images/34413687728781)

### 4. From the Presentation panel, choose the file you wish to download and select ***Enable download of the presentation***:

![](/images/34413702325005)

### 5. Back in the main area, select the Actions icon, and then the respective presentation file to be downloaded:

![](/images/34413687733517)

### 6. Once loaded, select ***Download presentation*** and chose where to save the PDF file:

![](/images/34413702327821)
