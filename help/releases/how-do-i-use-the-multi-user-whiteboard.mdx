## When the presenter enables the multi-user whiteboard, all users in the BigBlueButton session can annotate the whiteboard simultaneously.

To engage, as a presenter, click the multi-user whiteboard button on the toolbar.

<Frame>
  ![](/images/40564825113229.png)
</Frame>

When engaging the multi-user capabilities, the whiteboard controls will appear on the right-hand side in the presentation area. There are two toolbars that can be used when interacting with the whiteboard. The **styles** toolbar contains tools that allow you to change the style (color, dash pattern, size, and fill) of your annotations along with undo and redo tools. The **tools** toolbar contains tools that allow you to interact with your annotations (grouping, aligning, rotating, etc.) and create new annotations with selection, pen, eraser, shapes, arrows, text, sticky notes, and delete tools.

|  |  |
| --- | --- |
| Color | Select the color for your annotations to display in |
| Fill | Select objects that will be filled with a color |
| Opacity | Change the opacity of your object |
| Dash | Select the dash pattern of your object |
| Size | Select your font size (S)mall, (M)edium, (L)arge, or Extra Large (XL) |
