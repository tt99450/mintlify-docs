title: Data Formats for Alerts
description: JSON data formats for LMS analytics integration
---

When your account is configured with a data-url, it will authenticate using JWT (using the same shared secret) and return back a JSON of information on who is present in the session.

## Basic Course Roster

Here's a sample JSON to display the course roster:

```json
{
  "course": {
    "id": 5,
    "fullname": "Computer Applications (Fred)",
    "shortname": "CST8090-401",
    "groups": []
  },
  "enrollment": {
    "students": [
      {
        "id": 3,
        "firstname": "John",
        "lastname": "Doe",
        "groups": []
      },
      {
        "id": 8,
        "firstname": "Jane",
        "lastname": "Doe",
        "groups": []
      }
    ],
    "instructors": [
      {
        "id": 9,
        "firstname": "Fred",
        "lastname": "Dixon",
        "groups": []
      }
    ]
  },
  "session": {
    "notifications": []
  }
}
```
