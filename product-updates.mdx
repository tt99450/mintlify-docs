<Update label="February 2025" description="Released February 28, 2025">

## BigBlueButton 3.0

This is a major release of BigBlueButton and contains multiple new and improved features for teachers and students.

### New features

- **Improved chat.** Edit messages, add emojis, and reply with follow-up messages.
- **Improved whiteboard.** Rebuilt on tldraw 2.0.
- **Join audio with microphone.** Configurable — no more separate listen-only vs. microphone choice.
- **Away mode.** One click to set yourself away.
- **300 users in a room.** Extensive refatoring how supports 300 users in a single room.
- **Plugin support.** Plugins can now extend the functionality of the core. The Pick Random User capability now implemented as a plugin.


For the full list of changes, see the [v3.0.0 release on GitHub](https://github.com/bigbluebutton/bigbluebutton/releases/tag/v3.0.0).

</Update>
