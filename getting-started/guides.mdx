## Communication

| -------------------------- | ------------------ | ----------------------------------------------------------------------- |
| Talk to participants       | Microphone / Audio | [Join the Audio](/help/user-guide/join-the-audio)                       |

| Send a private message     | Private Chat       | [Chat Messages](/help/user-guide/send-public-and-private-chat-messages) |

| Set my status              | Status Indicator   | [Set a Status](/help/user-guide/access-the-options-menu#away-status)    |
