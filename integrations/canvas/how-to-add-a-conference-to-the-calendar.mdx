---
title: "How to add a conference to the Canvas calendar?"
---

When creating a BigBlueButton conference, there is an option within the conference to add it to the calendar.  However, this shortcut is not ideal

1. You must add everyone to the conference
2. You cannot specify many of the properties of a calendar entry, such as repeating.

A better approach is to first create the conference as you want (such as with specific users), then create the calendar entry and link directly to the conference.
