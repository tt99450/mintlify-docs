When you create or edit a BigBlueButton activity, Moodle shows the main settings page for the room.

## Main areas you may see

- General activity details
- Room settings
- Recording settings
- Participant and role settings
- Presentation content
- Schedule settings
- Common Moodle module settings

## Typical edit flow

1. Open the course.
2. Turn editing on.
3. Create a new BigBlueButton activity or edit an existing one.
4. Review the settings that match your teaching workflow.
5. Save the changes.

If you are starting from scratch, see [Create your first activity](/integrations/moodle/get-started/create-your-first-activity).
