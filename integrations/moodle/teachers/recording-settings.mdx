---

Some recording settings can be changed at the activity level by the owner or administrator of the BigBlueButton activity.

> These settings may be hidden unless an administrator has enabled them site-wide.

## Common recording settings

These settings may include:

- how the recordings table is displayed
- whether recordings from deleted activities are included
- whether only imported recording links are shown

## Recording table format

Some Moodle sites allow the recordings table to be displayed in a simpler HTML layout or a richer filtered view.

If you have many recordings tied to one activity, the richer table can be easier to search and scan.

## Deleted-activity recordings

When enabled, recordings that were created under activities later deleted from the course can still appear in the interface.

![Option to include recordings from deleted activities](/images/Screen_Shot_2018-10-10_at_12.37.17_PM.png)

## Imported recording links only

When enabled, the table can be filtered to show only imported recording links instead of every available recording for the activity.

![Option to show only imported recording links](/images/Screen_Shot_2018-10-10_at_12.37.27_PM.png)
