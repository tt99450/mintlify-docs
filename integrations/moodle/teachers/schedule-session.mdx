title: "Schedule a session"
description: "Schedule a BigBlueButton session in Moodle using open and close times so learners can only join during a specific window."
---

Moodle lets you schedule a BigBlueButton activity with specific open and close times.

## What scheduling does

Scheduling effectively controls when users can enter the room.

This is different from **Wait for moderator**:

- **Wait for moderator** delays entry until a moderator arrives
- scheduling can block entry before the room opens or after it closes

## Configure schedule settings

1. Open the activity settings.
2. Expand the schedule section.
3. Enable the schedule options you want.
4. Set a start time, an end time, or both.
5. Save the activity.

## Common patterns

You can configure:

- a start time only
- an end time only
- both a start and end time

## What users see

Before the room opens, enrolled users will not see the join button. Instead, Moodle shows that the conference has not started yet.

Once the room is open, invited users can join until the close time. After that, the join option is no longer available.

## Calendar behavior

When a session is scheduled, it can also appear in the course calendar view.
