BigBlueButton activities in Moodle also use the standard **Common module settings** section.

## What you can configure

- activity availability
- ID number
- group mode
- grouping

## Availability

If the activity is set to **show on course page**, students can access it subject to any other access restrictions.

If it is set to **hide from students**, the activity is available only to users who can view hidden activities.

## ID number

An ID number can help identify the activity for grading, reporting, or custom workflows.

![ID number field in common module settings](/images/Screen_Shot_2018-10-22_at_10.20.58_AM.png)

## Group mode

Group mode is defined at the course level by default, but the activity can also define its own mode unless the course forces a single setting.

Available modes include:

- **No groups**
- **Separate groups**
- **Visible groups**

![Group mode options in Moodle](/images/Screen_Shot_2018-10-22_at_10.20.21_AM.png)

## Grouping

Grouping is a collection of groups within a course. When grouping is enabled, students assigned to groups inside that grouping can work together within the configured activity scope.
