Recording-related settings can appear in both administrator and teacher workflows.

## Site-wide recording topics

- recording enabled by default
- display defaults for recording tables
- deleted-activity recording visibility
- imported recording visibility

## Activity-level recording topics

- whether the session can be recorded
- imported recording filters
- display options for the recordings table
