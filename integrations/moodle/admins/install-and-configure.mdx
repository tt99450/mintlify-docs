title: "Install and configure"
description: "Install the BigBlueButton Moodle activity and configure the BigBlueButton server URL and shared secret."
---

Administrators install BigBlueButton as a Moodle activity module and then configure the connection to the BigBlueButton server.

## Core setup steps

1. Install or upgrade the plugin through Moodle’s normal plugin workflow.
2. Open the BigBlueButton plugin settings in Moodle.
3. Enter the BigBlueButton server URL.
4. Enter the shared secret.
5. Save the settings.
6. Test by creating a sample activity and joining a room.

## Common configuration items

- BigBlueButton server URL
- Shared secret
- Checksum or signing options if available on your site
- Polling and recording refresh behavior

## After setup

Once the connection is working, instructors can begin creating activities in their courses.
