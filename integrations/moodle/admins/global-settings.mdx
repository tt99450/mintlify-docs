Administrators can control how BigBlueButton behaves site-wide through the plugin settings.

## Common global settings

These often include:

- server URL and shared secret
- default welcome message behavior
- recording defaults
- recording refresh behavior
- wait for moderator defaults
- guest access availability
- participant defaults
- presentation availability

## Why they matter

These settings determine:

- which teacher-facing controls are visible
- the default values used for new activities
- which advanced features are available on your site
