## As an Administrator or Owner of an activity you have the ability to edit recording names and descriptions.

A BigBlueButtonBN room/activity recording name or description can be edited once it has processed and is displayed in the recordings table for viewing.

![Screen_Shot_2018-10-11_at_1.45.45_PM.png](/images/Screen_Shot_2018-10-11_at_1.45.45_PM.png)

Locate the recording you'd like to edit and select theÂ **pencil icon** next to the recording name or description.

![Screen_Shot_2018-10-11_at_2.06.49_PM.png](/images/Screen_Shot_2018-10-11_at_2.06.49_PM.png)

Enter the text you'd like displayed into the recording name and description **text input fields**.

Be sure to press **enter** when finished or the changes will not save when the page is refreshed or when you navigate away from the recordings table.

![Screen_Shot_2018-10-11_at_2.02.53_PM.png](/images/Screen_Shot_2018-10-11_at_2.02.53_PM.png)

The edited name and description will appear in place of the originals and can be updated/edited as needed.
