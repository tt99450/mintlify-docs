## Administrator or owners can import recording links from other courses into a specific BigBluButtonBN room/activity recordings table.

By default, the BigBlueButtonBN plugin hides this option - it must be enabled by an administrator.

![Screen_Shot_2018-10-23_at_3.56.46_PM.png](/images/Screen_Shot_2018-10-23_at_3.56.46_PM.png)

To import recording links to an existing BigBlueButtonBN room/activity recordings table, navigate to the specific BigBlueButtonBN activity recordings table you would like to import recordings to.

Choose **import recording links.**

**![Screen_Shot_2018-10-23_at_3.58.08_PM.png](/images/Screen_Shot_2018-10-23_at_3.58.08_PM.png)**

Locate and select the course you would like to import recordings from.

***IT IS IMPORTANT TO NOTE: Only courses that have been enabled for import will display in the drop down.***

***![Screen_Shot_2018-10-23_at_3.58.14_PM.png](/images/Screen_Shot_2018-10-23_at_3.58.14_PM.png)***

Select from the list of available recordings which link(s) you would like to import.

![Screen_Shot_2018-10-23_at_3.58.24_PM.png](/images/Screen_Shot_2018-10-23_at_3.58.24_PM.png)

A pop up window will appear to confirm you would like to import the chosen link, choose **yes**.

![Screen_Shot_2018-10-23_at_3.58.31_PM.png](/images/Screen_Shot_2018-10-23_at_3.58.31_PM.png)

All imported recording links will appear under the existing activities' recordings in chronological order.
